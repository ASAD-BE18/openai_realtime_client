# OpenAI Realtime API Client for Python

## Installation

Install system deps:

```bash
brew install ffmpeg
```

Install python deps:

```bash
pip install -U openai pyaudio pynput pydub websockets
```

Set your openai key:

```bash
export OPENAI_API_KEY="sk-..."
```

## Usage

Run the interactive CLI:

```bash
python ./cli.py
```
